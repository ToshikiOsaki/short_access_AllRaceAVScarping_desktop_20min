2.  **必要なライブラリのインストール**
    リポジトリに `requirements.txt` という名前のファイルを作成し、以下の内容を記述してください。

    **`requirements.txt`**
    ```txt
    requests
    beautifulsoup4
    openpyxl
    lxml
    ```

    その後、ターミナル（コマンドプロンプト）で以下のコマンドを実行して、ライブラリを一度にインストールします。
    ```bash
    pip install -r requirements.txt
    ```
